# 系统使用手册

## 登录

访问 `http://localhost:8080/api/index.html`，使用 README 中测试账号登录，密码统一为 `123456`。

## 管理员

- 用户管理：查询用户、编辑用户、启用/禁用用户。
- 课程管理：查看系统课程，删除无选课关联课程。
- 公告管理：发布和维护公告。
- AI 助手：演示问答、推荐和摘要。

## 教师

- 课程管理：发布课程、查看课程。
- 选课名单：选择课程后查看学生名单。
- 成绩管理：录入和查看成绩统计。
- AI 助手：生成课程摘要、查询教学管理规则。

## 学生

- 课程中心：浏览课程、搜索课程、选课。
- 我的课程：查看已选课程并退课。
- 成绩管理：查看成绩。
- AI 助手：咨询选课流程、获取课程推荐。

## 常见问题

- 无法连接数据库：确认 MySQL 已启动，SQL 已导入，`application.yml` 密码正确。
- 端口冲突：修改 `server.port`。
- 前端依赖加载慢：页面使用 CDN，请确保浏览器可访问对应 CDN。
